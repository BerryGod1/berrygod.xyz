Status: **Working** PolSec is a free whitelist system (not public) which I made to avoid the price of luarmor. It prevents basic env tampering, function tampering, eq hooks, and jmps (these are really basic detections).

Features:
- Discord bot
- Automatic payments
- Multi script support
- Advanced analytics (IPs are not sent)

Security features:
- Dynamic responses
- Encryption
- Randomness is based on in-game values too
- Heartbeats (Doesn't support Solara)
- Basic tampering detections
